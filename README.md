# New Horizon Villas - Premium Vacation Rental Website

A luxury vacation rental showcase and booking platform for New Horizon Villas in Kea (Tzia), Greece. Built with modern web technologies and optimized for performance, accessibility, and SEO.

View your app in AI Studio: https://ai.studio/apps/e8249fb5-88fe-4632-a9de-9dc9a2ec0466

## ✨ Features

- **🌍 Bilingual Support**: Full English and Greek translations with context-aware content
- **📱 Responsive Design**: Mobile-first approach with optimized layouts for all devices
- **🎨 Premium UX**: Smooth animations, parallax effects, and luxury Mediterranean aesthetic
- **🔍 SEO Optimized**: Complete meta tags, Open Graph, Twitter Cards, structured data, and sitemap
- **♿ Accessible**: ARIA labels, semantic HTML, keyboard navigation, and WCAG compliance
- **⚡ Performance**: Code splitting, lazy loading, optimized images, and efficient bundling
- **🔒 GDPR Compliant**: Cookie consent banner and privacy policy
- **📄 Legal Pages**: Privacy Policy and Terms of Service
- **🎯 User-Friendly**: 404 page, error boundaries, and graceful error handling

## 🛠️ Tech Stack

- **Framework**: React 19 with TypeScript
- **Build Tool**: Vite 6.2
- **Styling**: Tailwind CSS 4
- **Routing**: React Router v7
- **Animations**: Motion (Framer Motion alternative)
- **Icons**: Lucide React
- **SEO**: React Helmet Async
- **Deployment**: GitHub Pages

## 🚀 Run Locally

**Prerequisites:** Node.js 18+

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser

## 📦 Build for Production

To build the app for production:

```bash
npm run build
```

To preview the production build locally:

```bash
npm run preview
```

## 🌐 Deploy to GitHub Pages

This app is configured to deploy to GitHub Pages at `https://emirsaffar-collab.github.io/New-Horizon/`.

### Automatic Deployment

The app automatically deploys to GitHub Pages when changes are pushed to the `main` branch via GitHub Actions workflow (`.github/workflows/deploy.yml`).

### Manual Deployment

1. Go to the "Actions" tab in the GitHub repository
2. Select the "Deploy to GitHub Pages" workflow
3. Click "Run workflow"

The app will be available at: https://emirsaffar-collab.github.io/New-Horizon/

## 🎯 SEO & Performance

- ✅ **Lighthouse Score**: Optimized for 90+ scores across all metrics
- ✅ **Meta Tags**: Complete Open Graph and Twitter Card implementation
- ✅ **Structured Data**: JSON-LD schema for vacation rentals
- ✅ **Sitemap**: XML sitemap with hreflang support
